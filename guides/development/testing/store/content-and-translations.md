# Content and translations

- Extensions must always be available in English. The German translation is optional. The description is available in English and German. If the extension is also available in other languages, it must be translated in Shopware as well.

- Allowed HTML tags: `<a>`, `<p>`, `<br>`, `<b>`, `<strong>`, `<i>`, `<ul>`, `<ol>`, `<li>`, `<h2>`, `<h3>`, `<h4>`, `<h5>`.

- There must be no advertising for your own services or contact information in admin, description, or images

- Links that encourage contact are not permitted. This also applies to support email addresses and backlinks.

In general, the extension's visual design must be consistent with that of Shopware and integrate seamlessly into the existing sections.

## Store description available (DE & EN)

### Description

- Minimum 200 characters
- Must describe functionality in detail
- Must be accurately translated 1:1 (EN ↔ DE)
- A precise and clear description of the extension and its use cases.

### Short description

- 150–185 characters
- Used as teaser and meta description

### Images

- The language used in the images must not be mixed
- The images should only show how the extension works
- At least one screenshot for the storefront and admin showing the features
- The images need to be high-quality

## Admin translations managed correctly

- English must be the fallback language and always be available
- This applies to all settings and error messages.

## More

**Own media folder**  

- Separate media folders must be created, or existing folders must be used to upload images.
- The basic structure of Shopware must not be changed or modified.
- Menu entries in the main menu of the Administration are not allowed.
