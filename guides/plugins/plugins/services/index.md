# Services

This section covers the service layer of plugin development. Services are where reusable plugin logic lives, dependency injection wires services together, and service decoration lets plugins change existing Shopware behavior.

- Register reusable plugin logic as a service: [Add Custom Service](./add-custom-service.md) |
- Inject Shopware or plugin services into another service: [Dependency Injection](./dependency-injection.md) |
- Change behavior of an existing Shopware service: [Adjusting a Service](./adjusting-service.md) |
