# Testing

## Unit Tests

[Vitest](https://vitest.dev/) is used for unit testing. The tests are located in the `__tests__` directory of each module.

## Running Tests

**Unit Tests**

```bash
pnpm run test
```

**Coverage**

  ```bash
  pnpm run test:coverage
  ```
