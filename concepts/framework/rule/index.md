# Rule

Shopware provides a generic **rule system** that allows you to describe business conditions as composable rules. These rules are evaluated against a specific constext, such as a cart, an order or a customer and are used accross multiple domains like checkout, promotions and flows.

On top if this rule system the **Rule Builder** is the Administration feature that lets users configure and combine rule conditions visially.

## Key Capabilities

- **Composable rule trees**

- **Context-aware evaluation**

- **Cross-domain usage**

- **UI configurable behavior control via Rule-Builder**

## Where Rules are used

- **Checkout and cart**

- **Promitions**

- **Flow Builder**

- **Product pricing**

- **Shipping and Payment methods**
